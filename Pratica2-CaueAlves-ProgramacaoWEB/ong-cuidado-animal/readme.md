# ONG Cuidado Animal

Este é um projeto simples, desenvolvido com HTML, que simula a página de produtos de uma **ONG Cuidado Animal**. O objetivo é demonstrar o uso de tags básicas do HTML, como cabeçalhos, parágrafos, imagens, vídeos e tabelas.

## Funcionalidades

O projeto atual inclui:

  * **Página de Produtos**: Exibe informações sobre os produtos da ONG.
  * **Imagens**: Mostra uma imagem de um petisco de ossinhos.
  * **Vídeo**: Reproduz um vídeo demonstrativo sobre os petiscos.
  * **Tabela de Informações**: Apresenta uma tabela detalhada com categoria, nome, descrição e preço dos produtos.

## Tecnologias Utilizadas

  * **HTML**: A estrutura principal da página é construída com HTML.
  * **CSS**: A página utiliza um arquivo externo chamado `estilos.css` para a estilização (embora o código CSS não esteja aqui, o HTML já está pronto para se conectar a ele).

## Como Visualizar

Para ver o projeto, você só precisa de um navegador web (como Google Chrome, Firefox ou Edge). Siga estes passos:

1.  Salve o código HTML em um arquivo com o nome `index.html`.
2.  Certifique-se de que os arquivos `estilos.css`, `petisco-ossinhos.png` e `petisco-videos.mp4` estão na mesma pasta que o `index.html`.
3.  Clique duas vezes no arquivo `index.html` ou arraste-o para a janela do seu navegador.

## Estrutura do Projeto

O projeto é composto pelos seguintes arquivos e pastas:

```
.
├── index.html
├── estilos.css
├── imagens/
│   └── petisco-ossinhos.png
└── videos/
    └── petisco-videos.mp4
```

  * `index.html`: O arquivo principal com o código HTML.
  * `estilos.css`: O arquivo de estilização (CSS).
  * `imagens/`: Pasta para armazenar as imagens do projeto.
  * `videos/`: Pasta para armazenar os vídeos do projeto.
